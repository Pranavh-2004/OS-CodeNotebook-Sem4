Some basic bash commands

1. To find shell being used:

```bash
echo $SHELL
```

example output:

```plaintext
/bin/zsh
```

2. To find terminal being used:

```bash
echo $TERM
```

example output:

```plaintext
xterm-256color
```

3. To find user:

```bash
echo $USER
```

example output:

```plaintext
pranavhemanth
```

4. To find present working directory:

```bash
echo $PWD
```

example output:

```plaintext
/Users/pranavhemanth
```

5. To find old present working directory:

```bash
echo $OLDPWD
```

example output:

```plaintext
/Users/pranavhemanth
```

```bash
echo $PATH
```

```bash
echo $HOME
```

example output:

```plaintext
/Users/pranavhemanth
```
